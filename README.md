# 🎨 Color Sense

A professional, high-precision color picker and analysis tool designed for creators and developers. **Color Sense** is a lightweight, pixel-perfect application that allows you to extract accurate color codes from any image with ease.

---

## ✨ Key Highlights

*   **🔒 Privacy First & Completely Offline:** No internet connection required. Your images never leave your device. Zero tracking, zero analytics, and zero data collection.
*   **⚡ Lightweight & Fast:** Optimized for speed and minimal storage footprint. It opens instantly and processes images smoothly.
*   **🎯 High Color Accuracy:** Advanced coordinate mapping logic ensures that the color you pick is exactly what's on the bitmap, down to the last pixel.
*   **📲 "Open With" Integration:** Seamlessly open images directly from your Gallery or any other app using the system "Open With" or "Share" menu.

---

## 🔥 Features

### 🕹️ Relative (Joystick-Style) Dropper
Unlike traditional pickers where your finger blocks the view, Color Sense uses a **relative movement system**. Slide your finger anywhere on the screen to move the crosshair precisely without obscuring the sampling point.

### 🔍 Advanced Zoom & Pan
*   **Gestures:** Use double-finger pinch to zoom and drag to pan.
*   **Clamped Boundaries:** The image is intelligently constrained so it never leaves the viewer area, ensuring you never "lose" your photo while zooming.

### 🎨 Premium UI/UX
*   **Material 3 Design:** Built with the latest Android design standards for a modern look and feel.
*   **Interactive Gradient Glow:** Buttons feature a dynamic, 360-degree gradient shadow that pulses and glows when pressed, providing satisfying tactile feedback.
*   **Stable Layout:** Switching between HEX, RGB, and HSL modes is smooth and doesn't cause any layout shifting.

### 🌗 Adaptive Dark Mode
*   Includes full support for Light and Dark themes.
*   Featuring a custom-designed Dark Mode toggle with a signature 135-degree rotated icon and gradient background.

## 🚀 How to Use

1.  **Launch the App:** Open "Color Sense" from your app drawer.
2.  **Pick an Image:** Tap the vibrant "Pick an Image" button or share an image from your gallery to the app.
3.  **Analyze:** Use the dropper to select colors. Tap the Copy icon to instantly save the HEX, RGB, or HSL code to your clipboard.
4.  **Customization:** Use the floating button at the bottom-right to toggle between Dark and Light modes.
